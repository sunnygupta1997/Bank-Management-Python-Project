# Bank Management System (Python)

## 📌 Project Overview
This project is a simple Bank Management System developed using Python. 
It allows users to perform basic banking operations like creating an account, depositing money, withdrawing money, and checking account balance.

## 🚀 Features
- Create bank account
- Deposit money
- Withdraw money
- Check account balance
- Input validation for invalid amounts

## 🛠 Technologies Used
- Python
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Install Python on your system
2. Download the project files
3. Open terminal or command prompt
4. Run the file using:

python bank_project.py

## 📚 Learning Outcome
- Understanding of Python basics
- Implementation of OOP concepts
- Basic banking logic using classes and methods
